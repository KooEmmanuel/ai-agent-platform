🤖 AI Agent Extension Test Page

📋 Test Instructions:

Step 1: Click the extension icon in your browser toolbar
Step 2: Login with your Drixai account (email/password or Google)
Step 3: Select an organization
Step 4: Select an agent
Step 5: A floating chatbot should appear on this page!

🔍 Debug Information:

Open Chrome DevTools (F12) and check the Console tab for debug messages.

Look for messages starting with:

  • 🤖 Agent selected
  • 📤 Sending message to tab
  • 📨 Content script received message
  • 🎯 FloatingChatbot.show() called
  • 🏗️ Creating chatbot UI
